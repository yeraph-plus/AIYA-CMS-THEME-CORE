WP Settings Framework Function Document

building up framework：

AYF::new_opt(array(
    'title' => 'AIYA-CMS',
    'slug' => 'options',
    'desc' => 'AIYA-CMS Theme',
    'fields' => array(
        array(            'title' => 'field name',
            'desc' => 'this is a field demo',
            'id'   => 'field_demo',
            'type' => 'switch',
            'default' => false,
        ),        array(            'type' => 'content',
            'desc' => 'this is a field demo',
        ),
    )
));

get settings:

AYF::get_checked('field_demo', 'options');

or

AYF::get_opt('field_demo', 'options');
